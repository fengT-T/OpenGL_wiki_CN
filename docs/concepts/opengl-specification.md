# OpenGL 规范（OpenGL Specification）

**OpenGL 规范**是完整描述 OpenGL 的官方文档，由 [OpenGL 架构评审委员会（ARB）](./opengl-arb.md)制定。每个 OpenGL 版本都有对应的规范文档。

## 文档特点

OpenGL 规范是相当复杂的文档，其目标读者是 **OpenGL 实现者**，而非普通用户。因此：

- 文档结构不够直观，难以作为学习工具使用
- 使用规范化的技术语言描述 API 行为
- 包含实现细节和边界条件说明

## 获取规范

OpenGL 规范 PDF 文档可在 [OpenGL Registry](http://www.opengl.org/registry/#apispecs) 获取，该网站同时提供所有 OpenGL 扩展规范。

## 规范内容

规范文档通常包含以下内容：

| 部分 | 说明 |
|------|------|
| 概述 | OpenGL 的基本概念和设计目标 |
| 核心 API | 所有核心函数的详细说明 |
| 状态机 | OpenGL 状态的组织和管理 |
| 渲染管线 | 图形管线的各个阶段 |
| 着色器 | GLSL 规范 |
| 扩展 | 可选功能的说明 |
