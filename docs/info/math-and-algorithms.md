# 数学与算法 (Math and Algorithms)

本节介绍 OpenGL 开发中常用的数学算法和技巧。

## 从窗口空间计算眼空间

**Compute eye space from window space**

将窗口空间坐标逆向变换到眼空间坐标的算法和着色器实现。常用于：

- 屏幕空间效果的后处理
- 从深度缓冲重建世界坐标
- 鼠标拾取计算

## 计算表面法线

**Calculating a Surface Normal**

OpenGL 光照计算需要顶点法线。本节介绍：

- 从顶点位置计算法线
- 平面法线与顶点法线的区别
- 法线平均和加权方法

## ID 缓冲

**ID-Buffer**

一种用于对象选择的技术，通过将对象 ID 渲染到专用缓冲区实现：

- 鼠标点击拾取
- 可视化调试
- 对象剔除判断

## 轨迹球

**Trackball**

实现虚拟轨迹球旋转的算法，用于：

- 模型交互旋转
- 相机轨道控制
- 平滑旋转动画

::: info 相关资源
这些算法在 3D 图形编程中广泛使用，理解其原理有助于优化渲染性能和实现复杂交互效果。
:::
